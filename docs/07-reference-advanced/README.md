### Q: Do I need to read this part sequentially?
**A:** No! This is reference material. Use it as needed:
- Quick lookups: Jump to specific sections
- Learning: Read through once, then refer back
- Troubleshooting: Go directly to your issue

### Q: Should I memorize all shortcuts?
**A:** No. Focus on these 5:
1. `Tab` - Accept suggestion
2. `Esc` - Reject suggestion
3. `Ctrl+I` - Inline chat
4. `Ctrl+Alt+I` - Chat view
5. `Ctrl+Enter` - See all suggestions

Learn others as you need them.

**A:** This guide is based on December 2025 documentation. Check official docs for latest updates:
